# Introducción a la Documentación

Bienvenido al repositorio central de documentación de **Marathon**.

En este espacio encontrarás información detallada, guías técnicas y manuales de referencia sobre los diversos proyectos tecnológicos implementados en la organización.

## ¿Qué encontrarás aquí?

Esta documentación está estructurada para cubrir las principales áreas de innovación y operación tecnológica:

*   **Desarrollo de Software**: Documentación sobre módulos personalizados, aplicaciones internas y arquitecturas de software.
*   **Automatización de Procesos**: Flujos de trabajo automatizados con herramientas como **N8N**, integraciones entre sistemas (ERP, CRM) y optimización de tareas repetitivas.
*   **Gestión de Sistemas**: Guías de administración para plataformas como **Odoo**, incluyendo configuraciones de permisos, creación de campos y mantenimiento.
*   **Infraestructura**: Detalles sobre el despliegue de servicios, uso de Docker y arquitecturas de servidores.

Nuestro objetivo es centralizar el conocimiento técnico para facilitar el mantenimiento, la escalabilidad y la mejora continua de nuestras soluciones.
